# 課題
* PHPブックマークアプリユーザー管理

# 課題の内容
* 以下の属性別に見ることのできるページが異なるブックマークアプリを作りました。
1. ログイン無し→ブックマークアプリの閲覧、登録が可能
2. ログイン有り→ブックマークアプリの閲覧、登録、編集、削除が可能
3. 管理者→ブックマークアプリの閲覧、登録、削除と、ユーザーの登録、更新、削除が可能
4. 非管理者→ログイン有りと同じ
